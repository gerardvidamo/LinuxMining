### Linux Mining Installation
Create dir for the repository
```console
$ sudo mkdir /usr/local/LinuxMining
```
cd to  `/usr/local` then clone repository
```console
cd /usr/local
$ git clone https://github.com/gerardvidamo/LinuxMining.git
```
Install
```console
$ bash install.sh
```
